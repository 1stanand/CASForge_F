# Gemini Review Scope

## Goal

Find why CASForge output quality is below target even though the pipeline works end-to-end.

## Focus

- parsing accuracy
- retrieval quality
- ranking/relevance
- prompt assembly
- final generation quality

## Do Not Focus On

- generic README improvements
- code style cleanup
- naming polish
- folder structure polish
- broad “review whole repo” commentary

## Expected Output

1. root-cause findings
2. prioritized issues
3. concrete fixes
4. which stage is most responsible for low accuracy
